linux内核，网络参数优化 sysctl.conf文件

Posted on 2017-06-03   |   In Linux   |  

配置文件/etc/sysctl.conf

禁用ip包转发

net.ipv4.ip_forward = 0
net.ipv4.conf.all.forwarding = 0

对直接连接的网络进行反向路径过滤

net.ipv4.conf.all.rp_filter = 1
net.ipv4.conf.default.rp_filter = 1

不允许接受含有源路由信息的ip包

net.ipv4.conf.all.accept_source_route = 0
net.ipv4.conf.default.accept_source_route = 0

达梦数据库的安装

首先说一下，较之人大金仓，个人感觉达梦数据库的技术文档更多也更全面，也很友好，同时也是国产为数不多的真正有自己技术的数据库产品。

1. 下载安装：

1.1 产品和文档下载：

可到达梦数据库官网下载Linux平台64位DM7试用版数据库，有一年的免费试用期，下载地址技术文档
主要文档有：DM Program.pdf、DM SQL.pdf、DM DBA.pdf

1.2 解压和安装：

下载到本地后，解压（tar -zxvf...），安装，可以根据解压文件夹下的DM_Install_zh.pdf安装文档—>“2.2Linux下DM服务器、客户端软件的安装”进行操作安装。

VMWare12-Ubuntu16.04学习

Posted on 2017-03-16   |   In Linux   |  

Ubuntu16.04 安装 Vmware-tools

建议安装 Ubuntu 自带的开源vmware-tools
$ sudo apt install open-vmware-tools-desktop

Ubuntu16.04 安装 sublime-text3

第一步：添加sublime text 3的仓库
$ sudo add-apt-repository ppa:webupd8team/sublime-text-3
按ENTER继续，建立信任数据库
第二步：更新软件库
$ sudo apt update
第三步：安装Sublime Text 3
$ sudo apt install sublime-text-installer
下面是sublime text 2的安装命令，与3稍有不同：
$ sudo add-apt-repository ppa:webupd8team/sublime-text-2
$ sudo apt-get update
$ sudo apt-get install sublime-text-dev
