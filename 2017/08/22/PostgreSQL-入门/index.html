Postgresql 安装

创建文件 /etc/apt/sources.list.d/pgdg.list
touch /etc/apt/sources.list.d/pgdg.list
增加一行 for the repository
deb http://apt.postgresql.org/pub/repos/apt/ trusty-pgdg main
Import the repository signing key, and update the package lists
wget --quiet -O - https://www.postgresql.org/media/keys/ACCC4CF8.asc | \ sudo apt-key add -sudo apt-get updateInstallsudo apt-get install postgresql-9.6`
安装完成后会默认：

  • 创建名为”postgres”的Linux用户
  • 创建名为”postgres”、不带密码的默认数据库账号作为数据库管理员
  • 创建名为”postgres”的表

Postgresql 常用命令

登录Postgresql Server
su postgres 切换到psotgres用户下
psql 登录到postgres数据库

列出所有的数据库：
\l or \list
列出所有用户：
\du
切换数据库：
\c dbname
列出当前数据库下的所有数据表：
\d
列出指定表的所有字段：
\d tablename
查看指定表的基本情况：
\d+ tablename
退出登录：
\q
