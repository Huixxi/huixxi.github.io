LaTeX Tutorial

Posted on 2018-09-08   |   In Tutorial   |  

Word is good, but LaTeX is perfect. Tired of Powerpoint? Then try Beamer
The first step to ‘Brexit’(Depart away from ‘O’ffice) is to learn and use LaTeX. So let’s get it.

Introduction

LaTeX: LaTeX is a high-quality typesetting system; it includes features designed for the production of technical and scientific documentation. LaTeX is the de facto standard for the communication and publication of scientific documents.
Beamer: Beamer is a LaTeX document class for creating slides for presentations.

Installation

So How to Install ‘LaTeX’ and ‘Beamer’.
The following options are mostly base on the first reference.
Install on Windows(Win8+,Win10):

  • First, you need to download the MikTex installer here. You can choose to download the entire 192.11MB installation package and just double click the .exe file to run the installation porcess. And you can also choose to download the Net Installer and let it download the components if you have a good net connection. When installing MiKTeX, choose “Yes” when asked “Always install missing packages on-the-fly”.
    If you alraedy have MikTex but not sure if the “Always install missing packages on-the-fly” is choosen, open the MikTex Console, then Settings –> General and check.

  • Next, you will need a typesetting environment. I recommend that you just use TeXWorks which comes with MikTex before you finish installing other packages such as the Beamer Class and make sure that your MikTex works well. After that, you can try to install other more human-friendly IDEs, like TeXNicCenter, TeXstudio which I prefered. Now let’s get back to the point.
    Creat a new file with your TeXWorks and paste the following:
    \documentclass{beamer}
    \begin{document}
    \end{document}
    When you compile this file, MikTex will help you to install the Beamer Class automatically.(WARNING:You may easily get some kind of errors. I will talk about it later.)
    That’s all I did to install ‘LaTex’ and ‘Beamer class’ on my computer. But there are still some continual process in the first reference and you can check it by yourself if you fail to finish the installation.

Deal with the Errors
I have met one nerve-wracking problem: Sorry, but pdftex.exe did not succeed and I fianlly found the solution here.
Here is what I did:

  • uninstalled my previous version of MiKTeX2.9
  • uninstalled my previous version of adobe reader
  • reinstalled the latest adobe reader
  • reinstalled my previous version of MiKTeX2.9 as above

There is an example:
Paste the following to a new file and compile it. If you have succeeded to install beamer, there is something like the image below with no errors.

\documentclass{beamer}

\usepackage[utf8]{inputenc}


%Information to be included in the title page:

\frame{\titlepage}

\end{document}

1. Algorithm Pseudocode

You can check this wikibook LaTeX/Algorithms. There is good example:
Code: 

\documentclass{article}
\usepackage[titlenumbered,ruled]{algorithm2e}
\begin{document}
\begin{algorithm}[H]
    \SetKwInOut{Input}{input}
    \SetKwInOut{Output}{output}

    \Input{A set of men $M$ such that for all $m \in M$, $m$ is free and has a preference list containing each woman $w \in W$ \newline 
    A set of women $W$ such that for all $w \in W$, $w$ is free and has a preference list containing each man $m \in M$}
    \Output{A perfect matching between $M$ and $W$ with no strong instability}
    \caption{Modified Gale-Shapley algorithm allowing indifferences}
\end{algorithm}
\end{document}
