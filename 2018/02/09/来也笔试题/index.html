4）解释进程与线程的区别： 

+ 进程：cpu分配资源的最小单位，各进程之间资源是不共享的，独立占有的。
+ 线程：cpu调度的最小单位，一个进程可包含多个线程，各线程之间共享同一资源

2.语法基础题

4）解释Python的变量作用域查找规则
Python的变量作用域查找顺序遵从：
local（局部）--> enclosing（函数范围）--> global（全局）--> build-in（内建对象）
5）解释Python的generator（生成器）的语法行为及适用场合

Generator适用于在内存资源有限，创建大型列表时元素又可以通过某种算法推算出来的情况下（比如斐波那契数列等等）
在Python中有两种方法实现generator:

  • 方法一：把python中的列表生成式的[]改为()

  • 方法二：函数定义中包含yield关键字,在每次调用next()的时候执行，遇到yield语句返回，再次执行时从上次返回的yield语句处继续执行。

6）请解释shallow-copy和deep-copy的区别，并用代码说明如何实现deep copy。
二者的差别仅体现在复杂对象上，比如list中包含list，或者class的实例.shallow-copy仅仅会copy第一层的对象，而deep-copy则会层层复制。

4.数据库或网络基础题

1）SQL中join类型(table_a, table_b)

+ Inner：生成一个包含a,b表相互匹配的记录的表
+ Leftouter：生成一个包含a中所有记录，以及b中所有与表a中对应的项，即使b中缺少与之相对应的某项，要用NULL填充。
+ Rightouter：生成一个包含b中所有记录，以及a中所有与表b中对应的项，即使a中缺少与之相对应的某项，要用NULL填充。
+ Full outer：生成一个包含a, b中所有记录，即使两表中缺少与之相对应的某项，要用NULL填充。
+ Cross：生成两表的笛卡尔积，不指定规则，两表中元素两两组合。
